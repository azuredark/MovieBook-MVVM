# MovieBook-MVVM
<p>
This app use <a href="http://www.omdbapi.com/">OMDB API</a> for movie datas. Also app works with MVVM. 
</p>

## Includes

+ 2 Model <code><a href="https://github.com/devmehmetates/MovieBook-MVVM/blob/main/MovieBook/Models/Movie.swift">Movie</a></code> <code><a href="https://github.com/devmehmetates/MovieBook-MVVM/blob/main/MovieBook/Models/MovieDetail.swift">Movie-Detail</a></code>
+ 2 Services <code><a href="https://github.com/devmehmetates/MovieBook-MVVM/blob/main/MovieBook/Services/DownloaderClient.swift">DownloaderClient</a></code> <code><a href="https://github.com/devmehmetates/MovieBook-MVVM/blob/main/MovieBook/Services/ImageDownloaderClient.swift">ImageDownloaderClient</a></code>
+ 2 ViewModel <code><a href="https://github.com/devmehmetates/MovieBook-MVVM/blob/main/MovieBook/ViewModels/FilmDetayViewModel.swift">MovieDetailViewModel</a></code> <code><a href="https://github.com/devmehmetates/MovieBook-MVVM/blob/main/MovieBook/ViewModels/FilmViewModel.swift">MovieArrayViewModel</a></code>
+ 3 View <code><a href="https://github.com/devmehmetates/MovieBook-MVVM/blob/main/MovieBook/Views/ContentView.swift">ListView(not refactored)</a></code>  <code><a href="https://github.com/devmehmetates/MovieBook-MVVM/blob/main/MovieBook/Views/CustomImage.swift">CustomImageView</a></code> <code><a href="https://github.com/devmehmetates/MovieBook-MVVM/blob/main/MovieBook/Views/DetailView.swift">DetailView</a></code>
+ 2 <code>ObservedObject</code>
